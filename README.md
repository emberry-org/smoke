<h2 align="left">
  <samp>
    <b>Development</b>
  </samp>
</h2>

To use this as dependency add ``smoke = { version = "0", git = "https://github.com/emberry-org/smoke" }`` to your projects ``Cargo.toml`` like so:
```
[dependencies]
smoke = { version = "0", git = "https://github.com/emberry-org/smoke" }
```
